## Description

REST API of Pick a Park project for Software Project Management exam's project of University of Camerino. 

## Requirements
- nodejs
- npm
- mongodb

## Running the app

### Start Backend

```bash
# move to backend directory
$ cd backend/
# install all packages
$ npm install
# run all tests (optional)
$ npm run test
# run backend
$ npm start
```

### Start Frontend
```bash
# move to frontend directory
$ cd frontend/
# install all packages
$ npm install
# run all tests (optional)
$ npm run test
# run frontend
$ npm start
```
